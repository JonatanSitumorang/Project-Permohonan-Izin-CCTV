# 🚀 Quick Start Guide - CCTV Permit System

## ⚡ Mulai dalam 5 Menit

### Langkah 1: Buka Aplikasi
Buka salah satu file di bawah ini langsung di browser:
- `index.html` - Formulir pengajuan
- `landing.html` - Halaman beranda

**Cara:**
1. Cari folder `cctv-permit-system`
2. Klik kanan pada file HTML
3. Pilih "Open with Browser" atau double-click

### Langkah 2: Buat Pengajuan
Dari halaman beranda atau langsung ke form:
1. Isi semua field yang diperlukan
2. Klik tombol "Kirim Pengajuan"
3. Dapatkan nomor referensi unik
4. Simpan nomor untuk tracking

### Langkah 3: Pantau Status
Kunjungi `submissions.html` untuk:
- Melihat semua pengajuan Anda
- Cek status pengajuan
- Lihat detail lebih lanjut
- Cetak dokumen

## 💡 Tips Penggunaan

### Isi Form dengan Benar
- ✅ Nama harus lengkap dan jelas
- ✅ Departemen/unit sesuai organisasi
- ✅ Lokasi spesifik (building, floor, room)
- ✅ Tujuan jelas sesuai pilihan
- ✅ No telepon dan email aktif

### Jika Pengajuan Ditolak
- 📞 Hubungi Tim TR atau Receptionist
- 📝 Tanyakan alasan penolakan
- 🔄 Buat pengajuan baru dengan revisi
- 📧 Catat feedback untuk referensi

### Pantau Pengajuan
- 📱 Buka submissions.html kapan saja
- 🔍 Cari pengajuan berdasarkan nomor referensi
- 📄 Cetak untuk dokumentasi
- ✉️ Hubungi jika tidak ada update setelah 24 jam

## 🔍 FAQ Cepat

**Q: Berapa lama pengajuan diproses?**
A: Biasanya 1x24 jam kerja untuk verifikasi awal.

**Q: Bagaimana cara cek status?**
A: Buka submissions.html dan lihat tabel pengajuan Anda.

**Q: Nomor referensi saya hilang?**
A: Semua nomor tersimpan di submissions.html. Cek di tabel.

**Q: Bisa ubah pengajuan yang sudah dikirim?**
A: Hubungi Tim TR untuk meminta perubahan.

**Q: Apa itu status "Menunggu Verifikasi"?**
A: Pengajuan sedang dalam review oleh Tim TR.

**Q: Kemana kirim dokumen tambahan?**
A: Hubungi Receptionist atau Tim IT untuk detail file submission.

## ⚙️ Troubleshooting Cepat

**Halaman tidak loading?**
- Refresh browser (Ctrl+F5 atau Cmd+Shift+R)
- Pastikan koneksi internet aktif

**Form tidak bisa di-submit?**
- Periksa semua field sudah terisi
- Lihat error message di browser (F12 → Console)

**Data hilang setelah tutup browser?**
- Data disimpan di localStorage
- Jika privacy mode aktif, data tidak persistent
- Gunakan browser normal untuk menyimpan data

**Nomor referensi tidak terlihat?**
- Kunjungi submissions.html
- Scroll ke bawah untuk lihat semua pengajuan
- Cek di tabel riwayat
