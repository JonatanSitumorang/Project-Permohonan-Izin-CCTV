═══════════════════════════════════════════════════════════════════════════════

💾 DATA MANAGEMENT
═══════════════════════════════════════════════════════════════════════════════

Penyimpanan: localStorage (Browser Local Storage)
Format: JSON array dari submissions

Struktur Data per Submission:
{
  "refNumber": "CCTV-2026-XXXXX",
  "nama": "Nama Pemohon",
  "departemen": "Nama Departemen",
  "lokasi": "Deskripsi lokasi",
  "tujuan": "Pilihan tujuan",
  "telepon": "No telepon",
  "email": "email@domain.com",
  "catatan": "Catatan tambahan",
  "timestamp": "7/1/2026, 12:30:45"
}

Nomor Referensi: Format CCTV-YYYY-#####
Contoh: CCTV-2026-03521
