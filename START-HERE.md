## ⚡ 3 Langkah Memulai (5 Menit)

### Langkah 1️⃣: Buka Aplikasi
```
Pilih salah satu:
📱 Double-click landing.html → Halaman beranda
📋 Double-click index.html → Form langsung
```

### Langkah 2️⃣: Isi & Kirim Form
```
1. Isi semua field dengan data yang benar
2. Klik "Kirim Pengajuan"
3. Catat nomor referensi: CCTV-2026-XXXXX
```

### Langkah 3️⃣: Pantau Status
```
📊 Buka submissions.html → Lihat riwayat pengajuan
🔍 Klik icon mata → Lihat detail
🖨️ Klik cetak → Export dokumen
```

## ❓ FAQ

**Q: Bagaimana jika saya lupa nomor referensi?**
A: Buka submissions.html → semua nomor tersimpan di tabel

**Q: Bisa ubah data setelah dikirim?**
A: Hubungi Tim TR atau buat pengajuan baru dengan revisi

**Q: Berapa lama proses approve?**
A: Biasanya 1x24 jam kerja untuk verifikasi awal

**Q: Bisa deploy di website sendiri?**
A: Ya! Buka DEPLOYMENT.md untuk panduan step-by-step

**Q: Ada biaya?**
A: Tidak, semuanya gratis dan bisa digunakan offline
